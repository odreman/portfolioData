# Portfolio de Visualización de Datos

Bienvenido a mi portafolio de visualización de datos. Este repositorio contiene una colección de proyectos que demuestran habilidades en análisis de datos, visualización y ciencia de datos utilizando Python y sus bibliotecas más populares.

## Proyectos

### 1. Análisis de Ventas Minoristas
**Directorio:** `retail_analysis`  
**Descripción:** Análisis de datos de ventas minoristas, incluyendo tendencias de consumo y comportamiento del cliente.  
**Tecnologías:** Python, Pandas, Matplotlib  
**Visualizaciones:** 
- Gráficos de barras horizontales para comparación de precios
- Gráficos de barras apiladas para análisis de ventas por categoría
- Gráficos de líneas para tendencias temporales
- Mapas de calor para correlaciones
[Ver proyecto](./retail_analysis/)

### 2. Análisis de Atletas Olímpicos
**Directorio:** `olympic_athletes_analysis`  
**Descripción:** Exploración de datos históricos de atletas olímpicos, incluyendo medallas y récords.  
**Tecnologías:** Python, Pandas, Matplotlib, NumPy  
**Visualizaciones:**
- Histogramas para distribución de edades y pesos
- Gráficos de barras agrupadas para comparación de medallas
- Gráficos de líneas para tendencias temporales
- Gráficos de dispersión para relaciones entre variables
[Ver proyecto](./olympic_athletes_analysis/)

### 3. Ajedrez y Estadísticas de Homicidios
**Directorio:** `chess_homicide_analysis`  
**Descripción:** Análisis de la relación entre partidas de ajedrez y estadísticas de homicidios.  
**Tecnologías:** Python, Pandas, Matplotlib, GeoPandas  
**Visualizaciones:**
- Gráficos de torta para distribución de victorias
- Histogramas con curvas de densidad
- Gráficos de barras apiladas
- Visualizaciones de distribución por género y raza
[Ver proyecto](./chess_homicide_analysis/)

### 4. Cerveza y Análisis de StackOverflow
**Directorio:** `beer_stackoverflow_analysis`  
**Descripción:** Estudio de la correlación entre recetas de cerveza y preguntas de programación.  
**Tecnologías:** Python, Pandas, Matplotlib, Seaborn  
**Visualizaciones:**
- Gráficos de dispersión para correlaciones
- Mapas de calor para matrices de correlación
- Gráficos de caja para distribución de datos
- Gráficos de barras para comparaciones categóricas
[Ver proyecto](./beer_stackoverflow_analysis/)

### 5. Natalidad y Restaurantes en España
**Directorio:** `spain_birthrate_restaurant_analysis`  
**Descripción:** Análisis de la relación entre tasas de natalidad y restaurantes con estrellas Michelin.  
**Tecnologías:** Python, GeoPandas, Matplotlib, Contextily  
**Visualizaciones:**
- Mapas coropléticos para visualización geográfica
- Gráficos de dispersión con líneas de tendencia
- Gráficos de barras para comparaciones regionales
- Visualizaciones de distribución espacial
[Ver proyecto](./spain_birthrate_restaurant_analysis/)

## Configuración

### Requisitos Previos
- Python 3.8+
- pip (gestor de paquetes de Python)
- Jupyter Notebook o JupyterLab

### Instalación
1. Clona el repositorio:
   ```bash
   git clone https://github.com/odreman/portfolioData.git
   cd portfolioData
   ```

2. Crea y activa un entorno virtual (recomendado):
   ```bash
   python -m venv venv
   source venv/bin/activate  # En Windows: venv\Scripts\activate
   ```

3. Instala las dependencias:
   ```bash
   pip install -r requirements.txt
   ```

## Uso

Cada proyecto contiene su propio Jupyter Notebook con el análisis completo. Para ejecutar un proyecto específico:

```bash
cd nombre_del_proyecto
jupyter notebook
```

## Tecnologías Utilizadas

- **Lenguajes:** Python
- **Análisis de Datos:** Pandas, NumPy
- **Visualización:** Matplotlib, Seaborn, GeoPandas
- **Geoespacial:** GeoPandas, Contextily
- **Notebooks Interactivos:** Jupyter
